# Stationery Store Management System

## Project Overview

The *Stationery Store Management System* is a C# .NET Console Application designed to manage stationery products in a store. The application allows users to securely log in, add and manage stationery items, purchase products, generate bills, monitor low stock, and sort inventory. This project demonstrates the implementation of Object-Oriented Programming (OOP) concepts, LINQ, Collections, Interfaces, Abstraction, Polymorphism, Inheritance, Encapsulation, and Custom Exception Handling.

---

## Features

- User Login Authentication (3 Attempts)
- Add New Stationery Items
- Display All Stationery Items
- Search Item by ID or Name
- Update Item Details
- Delete Item with Confirmation
- Purchase Item and Generate Bill
- View Low Stock Items
- Sort Items by Price, Name, and Quantity
- Custom Exception Handling
- In-Memory Data Storage using List Collection
