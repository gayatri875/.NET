# Automobile Vehicle Management System

## Overview
The Automobile Vehicle Management System is a C# Console Application developed to manage different types of vehicles for ABC Motors. It allows employees to add, view, search, update, delete, and manage vehicle records efficiently.

## Features
- Employee Login
- Add Vehicle
- View All Vehicles
- Search Vehicle
- Update Vehicle Price
- Delete Vehicle
- Calculate Vehicle Discount
- Show Vehicle Details
- Exit Application

## Vehicle Types
- Car
- Bike
- Truck

## Technologies Used
- C#
- .NET Console Application
- Object-Oriented Programming (OOP)
- Collections (List)
- LINQ

## OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Method Overriding

## Project Structure
- Vehicle.cs
- Car.cs
- Bike.cs
- Truck.cs
- VehicleManager.cs
- DiscountCalculator.cs
- Program.cs
- InvalidVehicleException.cs (Optional)

## Learning Outcome
This project demonstrates the implementation of CRUD operations, inheritance, polymorphism, collections, LINQ, and switch statements in a real-world console application.
