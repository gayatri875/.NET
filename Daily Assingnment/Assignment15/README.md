# Product Management System

## Description
A C# Console Application to manage products, search products, add products to a shopping cart, calculate discounts, and process payments.

## Features
- Add multiple products
- Display all products
- Search product by name
- Add products to shopping cart
- Validate product stock
- Calculate total amount
- Apply discounts
- Select payment method
- Display final bill

## Discount Rules
- Below ₹1000 → No Discount
- ₹1000 - ₹4999 → 10% Discount
- ₹5000 - ₹9999 → 20% Discount
- ₹10000 and Above → 30% Discount

## Payment Methods
- UPI
- Credit Card
- Debit Card
- Cash on Delivery
