# Customer Registration and Login System

## Description
A C# Console Application that allows users to register a customer account and then log in using their email and password. The system allows only three login attempts before locking the account.

## Features
- Register a new customer
- Store customer details
- Customer login
- Email and password validation
- Maximum 3 login attempts
- Account lock after 3 failed attempts

## Technologies Used
- C#
- .NET Console Application

## Concepts Used
- Classes and Objects
- Constructors
- Properties
- If-Else Statements
- While Loop
- Console Input/Output
