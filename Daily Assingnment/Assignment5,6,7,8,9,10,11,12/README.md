# Employee Management System

## Overview

This project is a C# Console Application developed to demonstrate Object-Oriented Programming (OOP) concepts such as Abstract Classes, Inheritance, Method Overriding, Collections, and Object Management.

## Features

* Create and manage employees.
* Support for Permanent and Contract employees.
* Assign leave balance based on employee type.
* Manage employee leave requests.
* Display employee and leave request details.
* Search employee by Employee ID.
* Display only Permanent Employees.
* Count total employees and leave requests.

## Technologies Used

* C#
* .NET Console Application
* Visual Studio / VS Code

## OOP Concepts Used

* Abstract Class
* Inheritance
* Method Overriding
* Encapsulation
* Polymorphism
* List<T>
* foreach Loop

## Project Structure

* Employee.cs
* PermanentEmployee.cs
* ContractEmployee.cs
* LeaveRequest.cs
* Program.cs

## Tasks Implemented

1. Created a List of Employees.
2. Displayed all employee details.
3. Created a List of Leave Requests.
4. Displayed all leave requests.
5. Displayed only Permanent Employees.
6. Searched employee by Employee ID (103).
7. Displayed the total number of employees.
8. Displayed the total number of leave requests.

## Output

The application displays employee information, leave request details, permanent employees, search results, and total counts using the console.
