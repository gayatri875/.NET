# Student Management System

## Overview

The Student Management System is a console-based application developed using C#. It demonstrates Object-Oriented Programming (OOP) concepts by managing students, courses, course registration, and fee calculation. The system provides an easy way to maintain student records, enroll students in courses, calculate fees, and display complete student information.

## Features

### Student Management

* Register a new student
* View all registered students
* Search a student by Student ID

### Course Management

* Add new courses
* View all available courses

### Course Registration

* Enroll students in one or more courses
* Prevent duplicate course registration
* Limit the maximum number of enrolled courses

### Fee Calculation

* Calculate fees based on total course credits
* Support different student types:

  * Regular Student
  * Scholarship Student
  * Part-Time Student

### Student Details

* Display student information
* Display enrolled courses
* Display total credits
* Display total fee

### Exception Handling

* Validates user inputs
* Handles invalid operations and runtime errors gracefully
